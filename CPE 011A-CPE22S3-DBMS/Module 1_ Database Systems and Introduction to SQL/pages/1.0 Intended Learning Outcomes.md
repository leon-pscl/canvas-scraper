Intended Learning Outcomes

At the end of the module, the students should be able to:

Explain the importance of databases, the different types of databases, and give its main functions and advantages

Compare Manual and Computerized File Systems to design database systems.

Analyze how the systems and database development life cycle can be used in designing database systems.

Demonstrate how to create, modify and delete database

Demonstrate how to insert, update and delete records from the database
