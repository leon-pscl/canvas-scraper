COURSE DESCRIPTION

The course introduces the students to the database and design concepts as part of a larger whole by placing them within the systems analysis framework and advanced database development, administration, management, security, and database implementation techniques. In this course, students create relational databases, write SQL statements to extract information to satisfy business reporting requests, create entity relationship diagrams (ERD) to design databases, and analyze table designs for excessive redundancy. As you develop these skills, you will use either MySQL or PostgreSQL to execute SQL statements and a database diagramming tool such as Visual Paradigm to create ERDs.

Course Code: CPE 011

Credits: 2 Units Laboratory

Contact Hours: 6 Hours Laboratory

Pre-requisites: CPE 009 - Object Oriented Programming

COURSE OBJECTIVE

The course aims to provide practical knowledge and skills in database management systems and how they are designed, implemented, and managed in an organization.  Students must present a database design project implemented and integrated into a software application solving a specific problem.

COURSE OUTCOMES

At the end of the course, the students should be able to:

Analyze existing database designs using fundamental database concepts and design rules (SO 7)

Create Entity-Relationship (ER) Diagrams based on specific requirements (SO 7)

Apply SQL Programming in database development (SO 7)

Apply advanced database techniques such as SQL Scripting, Stored Procedures, Stored Functions, Triggers in Database Development (SO 7)

Create a database design and application using any programming language for a specific client. (SO 5 and 7)

STUDENT OUTCOMES ADDRESSED BY THE COURSE

SO5 - Function effectively on a team whose members provide leadership, create a collaborative and inclusive environment, establish goals, plan tasks, and meet objectives.

SO7- acquire and apply new knowledge as needed, using appropriate learning strategies.

GRADING SYSTEM
The grades will be computed using the following formula:

Prelim Grade (PG)

PG = PCS (0.5) + PE (0.5)
PCS = Lab (0.35)  + Quiz (0.4) + Project (0.25)
PE = Lecture (0.5) + Lab (0.5)

Midterm Grade (MG)

MG = {MCS (0.5) + ME (0.5)}2/3 + 1/3(PG)
MCS = Lab (0.35)  + Quiz (0.4) + Project (0.25)
ME = Lecture (0.5) + Lab (0.5)

Final Grade (FG)

FG = {FCS (0.5) + FE (0.5)}2/3 + 1/3(MG)
FCS = Lab (0.25)  + Quiz (0.15) + Project (0.6)
FE = Lecture (0.5) + Lab (0.5)
