Here are the recommended IDEs that should be used for the class. If you already have an IDE installed that can run C++, then you're good to go!

Easy to Install:

Embarcadero Dev C++: [https://www.embarcadero.com/free-tools/dev-cpp](https://www.embarcadero.com/free-tools/dev-cpp) [A Dev C++ fork supported by modern developers]

[https://www.codeblocks.org/](https://www.codeblocks.org/)

CLion ([https://www.jetbrains.com/clion/](https://www.jetbrains.com/clion/)) [download using your university email]

Visual Studio Code (VS Code)

Download link: [https://code.visualstudio.com/](https://code.visualstudio.com/)

Using VS Code for C++ Programming: [https://code.visualstudio.com/docs/languages/cpp](https://code.visualstudio.com/docs/languages/cpp)

Installing mingw-w64: [https://www.msys2.org/](https://www.msys2.org/)

For students without a pc, for tablet/mobile usage, and for students who cannot install/run the other software on their machines:

Jdoodle: [https://www.jdoodle.com/online-compiler-c++/](https://www.jdoodle.com/online-compiler-c++/)

C++ Shell: [http://cpp.sh/](http://cpp.sh/)
