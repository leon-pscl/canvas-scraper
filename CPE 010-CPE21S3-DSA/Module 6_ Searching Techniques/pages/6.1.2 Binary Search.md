The binary search is fast and efficient, it also requires the data that will serve as elements to be sorted.

Search a sorted array by repeatedly dividing the search interval in half. Begin with an interval covering the whole array. If the value of the search key is less than the item in the middle of the interval, narrow the interval to the lower half. Otherwise, narrow it to the upper half. Repeatedly check until the value is found or the interval is empty.
Algorithm (Pseudocode)Step 1: low :=0, up=n-1
Step 2: Repeat while low <= up

//input the elements in ascending order
#include<iostream>
using namespace std;
int main()
{
int *a,no,up,i,low=0,f=0;

cout<<”\nENTER HOW MANY ELEMENTS TO BE STORED IN THE LIST”;
cin>>up;
a = new int[up];
//loop to input the elements into the array
for(i=0;i<up;i++)
{
cout<<”\nENTER A NUMBER”;
cin>>a[i];
}
cout<<”\nENTER THE NUMBER TO SEARCH”;
cin>>no;
for(i=(low+up)/2;low>=up;i=(low+up)/2)
{
if(a[i]==no)
{
f=1;
break;
}
else
if(a[i]>no)
up=i-1;
else
low = i+1;
}
if(f==1)
cout<<”\nTHE SEARCHING ELEMENT IS FOUND”;
else
cout<<”\nTHE SEARCHING ELEMENT IS NOT FOUND”;
}
