Introduction to Backtracking
Backtracking is an algorithmic technique where the goal is to get all solutions to a problem using the[brute force approach](https://www.baeldung.com/cs/brute-force-cybersecurity-string-search). It consists of building a set of all the solutions incrementally. Since a problem would have constraints, the solutions that fail to satisfy them will be removed.

It uses[recursive calling](https://www.baeldung.com/java-recursion)to find a solution set by building a solution step by step, increasing levels with time.In order to find these solutions, a search tree named[state-space tree](https://en.wikipedia.org/wiki/State_space_search)is used. In a state-space tree, each branch is a variable, and each level represents a solution.

A backtracking algorithm uses the[depth-first search](https://www.baeldung.com/cs/depth-first-search-intro)method. When it starts exploring the solutions, a bounding function is applied so that the algorithm can check if the so-far built solution satisfies the constraints. If it does, it continues searching. If it doesn’t, the branch would be eliminated, and the algorithm goes back to the level before.
