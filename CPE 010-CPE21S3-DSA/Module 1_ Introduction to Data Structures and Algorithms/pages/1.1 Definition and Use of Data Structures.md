What are Data Structures and Where are They Used?

They are used to represent the logical relationship that exists between individual elements of data.

It is a special way of organizing and storing data so that it can be used efficiently in a computer system.

Data structures are extremely important for programmers since it allows them to handle data efficiently through the appropriate use and implementation of various algorithms.

Simply put, the data structure is able to specify:

How the data is organized;

What are the accessing methods;

What is the degree of association; and

What are the alternatives for processing the information?

These data structures are the "building blocks" of the program.

It must be able to reflect those existing relationships that exist between data; and

It should be simple enough so that it can not only process data but also do it effectively.
