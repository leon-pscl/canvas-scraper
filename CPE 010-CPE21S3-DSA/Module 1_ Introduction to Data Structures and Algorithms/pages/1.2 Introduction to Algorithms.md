Brief History

What is an algorithm?
An algorithm is a given step-by-step procedure taken to solve a problem or specific kinds of problems. An algorithm is a computational module that takes input(s) and provides output(s).

Important notes:

The pre-arranged steps must be well-organized and well-defined in sequence from input to output.

Only accurate and truthful if expected outputs are provided.

Efficiency is dependent on space and time complexities.
Steps Required to Develop an Algorithm
Find a method for solving a problem.

The steps that comprise the algorithm must be well-defined.

A pseudocode is useful in universally representing these steps.

Validate the algorithm.

All steps in the algorithm must be validate, provide input and check if the expected output is provided.

Provide input with an expected output to test its accuracy.

Implement the algorithm.

Algorithms are not usually language dependent.

Use your preferred programming language to implement it.
