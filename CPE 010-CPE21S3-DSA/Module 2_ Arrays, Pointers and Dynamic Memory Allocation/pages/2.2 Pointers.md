Pointers
Pointers are symbolic representations of addresses. They enable programs to simulate call-by-reference as well as to create and manipulate dynamic data structures. Its general declaration in C/C++ has the format:

Syntax:
datatype *var_name;
int *ptr;   //ptr can point to an address which holds int data

Source: Abhirav Kariya
How to use a Pointer?
Define a pointer variable by using an asterisk (*). The following are valid variations:

int *ptr;

int * ptr;

int* ptr;

Assign the address of the variable to a pointer using the unary operator (&).

Access the value in the address by using the asterisk (*) which returns the value of the variable located at the address specified by the operand.
