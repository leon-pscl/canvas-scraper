Doubly Linked Lists
Traversed in either direction. It is a linked list in which every node has a next pointer and a backpointer.

Every node contains address of next node

Except last node

Every node contains address of previous node

Except the first node

Typical operations:

Initialize the list

Destroy the list

Determine if list empty

Search list for a given item

Insert an item

Delete an item, and so on
