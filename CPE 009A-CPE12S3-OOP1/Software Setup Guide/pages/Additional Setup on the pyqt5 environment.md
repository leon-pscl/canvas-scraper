Instructions [Step-by-step for Windows]
To be able to use the pyqt5 with Spyder in this environment we will go to home. Observe that the Spyder IDE and Jupyter Notebook says install instead of Launch (unlike before).

Click install on Spyder IDE because the Spyder IDE in the base environment is different from the pyqt5 environment. Basically, Spyder IDE does not exist in our newly created environment.

Wait for Spyder IDE to finish installation. You will now see a new IPython Console or Qt Console and Spyder IDE with the Launch button.

You may now go back to Environments and click Base to switch back to Base Environment. Wait for the base (root) environment to load.

After it has finished, you may go to Home. Observe the number of software with Launch button available than pyqt5 (just to see the difference between the two environments)

For convenience, you may switch between the two environments easily using the Dropdown menu on Applications On , you can select between base and pyqt5 anytime.

Return the environment back to base (root) environment as this will be the environment we'll use for modules 1-3. Then switch to pyqt5 environment at Module 4.

Congratulations! You're done with the setup.
