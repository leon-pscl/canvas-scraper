This module is intended to help students who haven't yet installed Anaconda on their machine and need help or guide on how to install Anaconda as well as the system requirements.

This module also shares some of the software that we'll be using that are pre-packaged when we install the Anaconda Distribution.

For students that already have Anaconda Installed on their machine, you might just want to go over the modules as a review and also verify whether you have the required software and package that we'll be using for this tutorial.
