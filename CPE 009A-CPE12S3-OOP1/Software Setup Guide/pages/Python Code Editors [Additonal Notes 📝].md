Python can be accessed/used in many ways. Depending on the project you're working on, you may prefer one method over the other. Therefore familiarity with the different methods and choosing between those options based on your need is essential.

Ways to run Python Programs

Directly Accessing the Python Interpreter (Python Shell)
> In Python, we can directly run our code using the Interpreter or the Python Shell. This can be accessed in various ways, depending on whether you installed Python itself or installed Anaconda (Python + package and software bundles). Python programs made in the Shell does not get saved but allows you to quickly test out pieces of your code.

Using a Code Editor (IDE)

Note: These local code editors (Spyder IDE, Visual Studio Code or VS Code, and Jupyter Notebook) is bundled with the Anaconda Distribution.
