Instructions [Step-by-step for Windows]
On your computer, open the Anaconda Navigator

Click on Environments and Click Create at the bottom. A small window should pop-up afterwards.

Set the name as pyqt5 (though other names are allowed like tutorial). Do not change the other settings then click Create and wait for the environment to finish.

You should see the pyqt5 below the base/root environment.

Click the play button (triangle button pointing to the right) and click Open Terminal. A command prompt (cmd) should open.

Type the command pip install pyqt5

Note: The commands are case sensitive.

Wait for Command Prompt to indicate that pyqt5 was successfully installed.

Verify this by typing the command pip list

Congratulations, we have now installed the pyqt5 package to our computer. Proceed to the next module: [Additional Setup on the pyqt5 environment](https://tip.instructure.com/courses/56131/pages/additional-setup-on-the-pyqt5-environment)to finalize the setup of this environment.
