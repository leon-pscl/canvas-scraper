Instructions [Step-by-step for Windows]
Open your Web Browser.

Type in the URL this link: [https://www.anaconda.com/products/individual](https://www.anaconda.com/products/individual)

Once the Anaconda page has loaded, click the download button.

Under Windows, select the 64-bit Graphical Installer and it download the installer. You may select 32-bit installer if you have a 32-bit machine.

Once the download has finished, run the installer and click Next.

Click I agree.

Do not change the Default Setting (Just me should be selected) then click Next.

Do not change the default installation path and click Next.

Do not change the default configuration and click Next.

The installation will now start, please wait until the installation has finished.

Once the installer has indicated Complete, you may click next

Click next.

Make sure to uncheck all the options, then click Finish. The installation window will now close.

To verify, whether Anaconda was successfully installed and is working. Open your Start Menu and browse Recently Added programs. Click on Anaconda Navigator.
Note: You can also access Anaconda Navigator by using search. (Windows Key + F then type Anaconda Navigator)

The Anaconda Icon will now load in the middle of the screen. Please wait for the Anaconda Navigator to load.

Congratulations, you have now successfully installed Anaconda. You may click "Ok, don't show again" to dismiss the Welcome window.

Anaconda Navigator is now installed.
