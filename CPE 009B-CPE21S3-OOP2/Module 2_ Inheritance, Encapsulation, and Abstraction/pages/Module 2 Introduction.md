Welcome to this module everyone. This module is intended to familiarize you with the concepts of the core Object-Oriented Programming Principles Inheritance and its levels, Abstraction, and Encapsulation.

At the end of this module, you should have and be able to do the following:

1.         Understand the concepts of inheritance, abstraction and encapsulation using Python

2.         Differentiate single, multiple and multilevel inheritance.

3.         Develop a program using different types of inheritance, abstraction, and encapsulation.
