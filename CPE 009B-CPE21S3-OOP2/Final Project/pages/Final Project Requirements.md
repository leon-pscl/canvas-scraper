The final project in CPE 009 Object-Oriented Programming (OOP) is a Desktop Application using PyQt5 that is related to Science Technology Engineering Mathematics (STEM) that could be any of the previous related subjects you have taken or just related to STEM in general.

The desktop application is required to be developed in Python using the PyQt5 Framework and must at least support a small database which maybe any of the following lightweight databases that support dictionary based storage and calls:

sqlitedict (most recommended)

pickledb

tinydb

The project is to be done with 5 members and the assigned group once finalized cannot be changed except for valid reasons.

There will be additional +15 points for any pair/individual that will develop their project using Git (Version Control System) and Github/Gitlab platform.
