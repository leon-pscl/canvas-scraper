Course Code and Name
CPE311 - Computational Thinking with Python

Course Description
The course introduces students to the principles of computational thinking as a way of solving problems, designing systems, developing algorithms, and understanding human behavior. This course explores making use of different levels of abstraction, decomposition, modularization, and recursion to understand and solve problems more effectively. It uses mathematical concepts such as induction, to develop algorithms and produce more efficient, fair, elegant, and secure solutions. The object-oriented language, Python, will be used to demonstrate computational thinking of students by solving real-life problems.
Credits
3 units  (2 units lecture, 1 unit laboratory)
Prerequisites
MATH 013 – Linear Algebra with Matlab

MATH 019A – Engineering Data Analysis

Course Objectives

This course aims to provide students with an understanding of the role computation can play in solving problems. It also aims to show students techniques and methods on writing application programs that allow them to solve simple problems using the Python programming language. In addition, students will also learn how to visually represent the analysis of the problems and its solutions using built-in libraries for analysis and visualization.
Course Intended Learning Outcomes (CILOs)
By the end of the course, the students will be able to:

1. Understand the applications of mathematics in the real world setting. (SO a)

2. Explain how to apply a systematic approach to problem-solving and apply modelling as part of the problem solving.  (SO a)

3. Perform testing of solutions as a whole through system testing and acceptance testing. (SO d, e)

4. Apply Python programming language to analyze data, solve a problem and present it to the stakeholders. (SO a, d, e, j, k )
