WHAT IS PROGRAMMING LANGUAGE

A programming language is a set of rules that provides a way of telling a computer what operations to perform.

A programming language is a set of rules for communicating an algorithm. It provides a linguistic framework for describing computations

A programming language is a notation system for describing computation in a machine-readable and human-readable form. A programming language is a tool for developing executable models for a class of problem domains.

English is a natural language. It has words, symbols, and grammatical rules.

1.  A programming language also has words, symbols, and rules of grammar.
2. The grammatical rules are called syntax.
3. Each programming language has a different set of syntax rules.

WHAT ARE THE TYPES OF PROGRAMMING LANGUAGES?

FIRST GENERATION LANGUAGES

* Machine language

* Operation code – such as addition or subtraction

* Operands – that identify the data to be processed.

* Is machine-dependent as it is the only language the computer can understand.

* Very efficient code but very difficult to write.

SECOND GENERATION

* Assembly languages

* Symbolic operation codes replaced binary operation codes.

* Assembly language programs needed to be “ assembled” for execution by the computer. Each assembly language instruction is translated into one machine language instruction.

* Very efficient code and easier to write.

THIRD GENERATION

* Closer to English but included simple mathematical notation.

* Programs written in source code must be translated into machine language programs called object code.

* The translation of source code to object code is accomplished by a machine language system program called a compiler.

* Alternative to the compilation is the interpretation which is accomplished by a system program called an interpreter.

*Common third-gen. Languages

* Fortran, COBOL, c and c++, visual basic

FOURTH GENERATION

* A high-level language (4gl) that requires fewer instructions to accomplish a task than a third gen. Language.

* Used with databases
