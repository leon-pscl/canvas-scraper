The grades will be computed using the following formula:

Class Standing Components
Assignment -10%

Seatwork/Lab Activity– 30%

Written Quiz/Practical Assessment - 60%

Prelim Grade (PG)

PG = PCS (0.5) + PE (0.5)
PCS = Prelim Class Standing
PE = Prelim Exam

Midterm Grade (MG)

MG = {MCS (0.5) + ME (0.5)}2/3 + 1/3(PG)
MCS = Midterm Class Standing
ME = Midterm Exam

Final Grade (FG)

FG = {FCS (0.5) + FE (0.5)}2/3 + 1/3(MG)
FCS = Final Class standing
FE = Final Exam
